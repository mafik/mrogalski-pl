Dobre hasła

Złożone hasła (małe i duże litery, cyfry, interpunkcja), entropia w okolicach 2^95.

import random, string
z = string.ascii_letters + string.digits + '$&=*+!#~%?^@-_;:<>'
n = 15 # 8
p = ''.join(random.choice(z) for i in range(n))
print(p)

Jeśli ktoś jest na tyle szalony by stosować je na co dzień, polecam program do zarządzania zbiorem haseł (Portfel pod KDE, Seahorse pod Gnome, Wallet pod Androidem).

Mniej dobre hasła

Tworzą rymowane słowo, proste w fonetyzacji. Po usunięciu dużych liter, łatwe do przekazania drogą głosową lub wstukania na klawiaturze telefonu.

from random import choice as ch
c = 'pfgcrlddhtnszvwmbxkjq'
v = 'aoeuiy'
a = ch(c) + ch(v) + ch(c) + ch(v)
b = ch(c) + ch(v) + a[2:]
p = a.capitalize() + b.capitalize()
print(p)
